# Contributing to Claude Code Daily

Thank you for your interest in contributing! This project thrives on community knowledge.

## How to Submit a Tip

### Option 1: Open an Issue

1. Go to [Issues](https://github.com/mrsarac/claude-code-daily/issues)
2. Click "New Issue"
3. Use this template:

```markdown
## Tip Title
[Short, descriptive title]

## Category
- [ ] Orchestration
- [ ] Context Management
- [ ] Workflow
- [ ] Subagents
- [ ] Tooling

## Description
[Explain the technique in 2-3 sentences]

## Code Example (if applicable)
```bash
# Your code here
```

## Source
[Your name/handle, or original source with link]
```

### Option 2: Pull Request

1. Fork this repository
2. Add your tip to the appropriate category file in `tips/categories/`
3. Follow the existing format
4. Submit a PR with title: `Add tip: [Your Tip Title]`

## Tip Guidelines

### ✅ Good Tips
- Specific and actionable
- Include code examples when possible
- Properly attributed to source
- Tested and verified

### ❌ Avoid
- Vague or generic advice
- Tips that are already covered
- Unattributed content from others
- Platform-specific tips (unless clearly labeled)

## Code of Conduct

- Be respectful and constructive
- Give credit where due
- Help others learn
